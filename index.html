ICBC Bot

To use the Bot, text commands to +1 (604) 229 2696. The possible commands are:

Start the Bot

Provide the information you use to sign into ICBC in a text message with the following format:

Start lastname licensenumber keyword location

Change earliest or latest date to check for an appointment

Send a text to change the earliest date to check for appointments:

Earliest mm/dd/yyyy



Send a text to change the latest date you want to check for appointments:

Latest mm/dd/yyyy

Check that the Bot is using the correct info

Check info

Update info

Update lastname licensenumber keyword location

Stop the Bot

Finish
